## Clone Repository

```bash
git clone <repository_url>
cd svacs_unified_core
```

## Create Virtual Environment

### Windows

```bash
python -m venv .venv
.venv\Scripts\activate
```

```bash
python3 -m venv .venv
source .venv/bin/activate
```

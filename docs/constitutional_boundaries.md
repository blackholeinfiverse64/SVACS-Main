# CONSTITUTIONAL BOUNDARIES — NICAI

## Purpose

NICAI operates as a replay-safe maritime intelligence participation layer inside the unified SVACS + TANTRA ecosystem.

NICAI does not execute sovereign authority.  
NICAI only generates deterministic intelligence outputs with complete observability and replay continuity.

---

# 1. WHAT NICAI OWNS

NICAI owns:

- Maritime signal processing
- Perception event generation
- Intelligence event generation
- Validation propagation support
- Trace continuity generation
- Telemetry emission support
- Replay-safe lineage reconstruction
- Deterministic orchestration visibility
- Append-only event participation

NICAI responsibilities are limited to intelligence participation and observability support.

---

# 2. WHAT NICAI DOES NOT OWN

NICAI does NOT own:

- Sovereign execution authority
- Governance authority
- Operational authorization
- Enforcement decisions
- Semantic truth authority
- Final execution approval
- Token issuance authority
- State override permissions
- Dashboard execution control

NICAI cannot independently execute or authorize operational actions.
